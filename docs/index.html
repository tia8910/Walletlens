WalletLens — Free All-Asset Portfolio Tracker

See every asset in one lens — crypto, US stocks, gold, silver, fiat & FX. 100% free. No account. No data collection. Your portfolio is stored only in your browser.

Features

  • Live prices for crypto, stocks, gold, silver, fiat — all in one dashboard
  • AI portfolio analysis: health score, Sharpe ratio, max drawdown, P&L attribution
  • Stress-test scenarios for BTC ±10% / ±20%
  • Multi-target sell plans with live progress bars
  • Live whale tracker and market-cap movers
  • Hide-values privacy mode and shareable wallet card
  • Backup as a single gzip-compressed code — no cloud needed
  • Installable Progressive Web App, works offline-first

Frequently asked questions

Is WalletLens free?

Yes — 100% free, no account, no paid tiers, no ads in the portfolio view.

Does WalletLens track stocks and gold?

Yes. Crypto (CoinGecko / Binance / CoinCap), US stocks (Stooq), gold and silver (Gold-API), fiat via live FX rates, plus user-defined bonds and other assets.

Where is my data stored?

Only in your browser's localStorage. No server, no account, nothing leaves your device. You can export a single backup code to move between devices.

Can I install WalletLens as an app?

Yes — it's a PWA. Install it on iOS, Android, or desktop and it opens offline-first after the first visit.
